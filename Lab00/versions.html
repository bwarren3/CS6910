Versions of Macala

Versions of the game date back to the 7th century and evidence suggests the game existed in Ancient Egypt

The name is a classification or type of game, rather than a specific game. There are many different mancala games here are a few:

  • Alemungula
  • Kalah
  • Dakon
  • Sungka
  • Hoyito
  • Many more!

Although more than 800 names of traditional mancala games are known, some names denote the same game, while some names are used for more than one game. Almost 200 modern invented versions have also been described.
